1️⃣ System Initialization Tests
✅ TC-01: Valid System Initialization

Description:
التأكد إن النظام بيقرأ عدد المعادلات ويخزنهم صح.

Input:

3
2x1+3x2+4x3=16
1x1+2x2+1x3=8
3x1+1x2+2x3=13

Expected Result:

يتم تحميل النظام بنجاح.

يتم استخراج 3 متغيرات (x1, x2, x3).

لا يظهر أي Error.

❌ TC-02: Equation Without '='

Input:

2
2x1+3x2+4x3
1x1+2x2+1x3=8

Expected Output:

Invalid equation format. Each equation must contain exactly one '=' sign.
Example of valid format: 2x1+3x2=16
❌ TC-03: Equation With Multiple '='

Input:

1
2x1+3x2==16

Expected Output:

Invalid equation format. Each equation must contain exactly one '=' sign.
